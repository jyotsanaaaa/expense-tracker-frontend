## Project Overview

This is the frontend of the Expense Tracker application built using Next.js (React).

It allows users to:
- Add expenses
- View expenses
- Edit expenses
- Delete expenses
- View a dashboard with charts
- Convert currency

---

## Tech Stack

- Next.js (React)
- Axios
- Recharts
- Vercel (Deployment)
